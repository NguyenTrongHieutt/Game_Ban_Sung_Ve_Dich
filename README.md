# Game
The game builds characters to overcome obstacles, destroy enemies and bosses to reach the finish line.
## Main Player
Use `A`, `D` to move.\
Use `J` to jump.\
Use `K` to shoot bullets.\
Use `L` to change bullets.\
Use `I` to place the wall.\
Use `O` to turn on the shield.\
